# Linexin Package Archive

Browsable archive of past Linexin repository packages, organized like [archive.archlinux.org](https://archive.archlinux.org/packages/).

## Structure

```
x86_64/
└── <first-letter>/
    └── <YY_MM>/
        ├── <package>-<version>-<arch>.pkg.tar.zst
        └── <package>-<version>-<arch>.pkg.tar.zst.sig
```

Browse by navigating to `x86_64/<letter>/<date>/` to find archived package versions.
